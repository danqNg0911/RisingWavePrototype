# Official Benchmark Track

This repository does not attempt to reproduce official benchmark numbers locally.

To reproduce the official benchmark methodology separately:

```bash
git clone https://github.com/risingwavelabs/nexmark-risingwave-1.0.git
cd nexmark-risingwave-1.0
```

Then follow the upstream README and prerequisites for Kubernetes, Helm, and the RisingWave operator.
